# Consultancy Website

A modern, professional homepage for NEXGEN Tech Consultancy. Built with pure HTML and CSS to showcase technology consulting services, expertise, and previous successful projects.

## Features
- **Premium Design:** Curated dark aesthetics and typography that conveys a professional and modern image.
- **Dynamic Interactions:** Hover effects and micro-animations to enhance user experience.
- **Responsive Layout:** Works seamlessly across desktops, tablets, and mobile devices.
- **Content Strategy:** Highlighting Technology & Innovation explicitly (AI, IoT, Automation, etc.).

## Setup Instructions
To view or work on this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/KALIFFATHULLAHS/consultancy_website.git
   ```
2. Open `index.html` in any modern web browser to view the site.
3. Edit `index.html` and `styles.css` using any text editor or IDE (such as VS Code).

## Technologies Used
- HTML5
- CSS3
